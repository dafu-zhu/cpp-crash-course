# C++ Crash Course

Interactive study guide for UChicago FINM 32600: Computing for Finance.

1. **C++ Foundations** — Types, Functions, References, Pointers
2. **OOP & Classes** — Classes, Templates, STL Containers
3. **Memory Management** — Special Members, Smart Pointers, RAII
4. **I/O, Algorithms, Errors** — File I/O, STL Algorithms, Exceptions
5. **Inheritance & Pricing** — Polymorphism, Black-Scholes, Monte Carlo, Binomial Trees
6. **AI, RL & Trading** — Eigen, Reinforcement Learning, FIX Protocol, VaR

Based on lectures by Prof. Chanaka Liyanarachchi.
