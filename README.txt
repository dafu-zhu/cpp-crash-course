=============================================
  C++ CRASH COURSE — How to Run on Windows
=============================================

STEP 1: Install Node.js (if you don't have it)
   → Go to https://nodejs.org
   → Download the LTS version
   → Run the installer (click Next through everything)

STEP 2: Open this folder in Command Prompt
   → Open File Explorer, navigate to this folder
   → Click the address bar, type "cmd", press Enter
   (This opens Command Prompt in this folder)

STEP 3: Run these two commands:
   npm install
   npm run dev

STEP 4: Open your browser
   → It will print something like:
     Local: http://localhost:5173/
   → Hold Ctrl and click that link, OR
   → Open your browser and go to http://localhost:5173

STEP 5: To stop the server
   → Go back to Command Prompt
   → Press Ctrl+C

That's it! The app runs in your browser.
